## Firebase Features

- Authentication: đăng nhập/đăng ký bằng Firebase Auth
- Firestore: lưu `users`, `movies`, `theaters`, `showtimes`, `tickets`
- Notification: nhắc giờ chiếu bằng notification local và FCM service
